# Cyhermes Dashboard

## Prerequisites

Before you begin, ensure you have met the following requirements:

    * You have installed the latest version of node & npm

## Installing

    ```
    npm install
    ```

## Development use

    ```
    npm start
    ```

## Deployment

    ```
    npm run build
    ```

## Built With

    * Create-React-App
    * Sass
    * Gsap
    * Ant Design
